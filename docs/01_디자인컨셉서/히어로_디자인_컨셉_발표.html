CONCEPT A · 데이터 증식

데이터 증식 Data Multiplication

세 키워드를 하나의 오브제로 모은다. 원본 큐브 하나가 여러 개로 증식했다가 다시 하나로 모이는 메탈릭 데이터 큐브.

확장 = 증식

큐브가 여러 개로 갈라지며 늘어나는 모습이 데이터 확장을 직역한다.

신뢰 = 회귀

늘어난 큐브가 다시 하나로 모여, 하나의 공공 출처에서 비롯됨을 보인다.

명료 = 정돈

정육면체의 깔끔한 격자와 메탈릭 질감이 정제된 데이터의 명료함을 준다.
