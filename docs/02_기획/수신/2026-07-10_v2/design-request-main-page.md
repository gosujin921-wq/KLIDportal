# 메인 페이지 정적 디자인 요청서

> **작성일**: 2026-07-03
> **대상**: AI 영상학습 사용자 포털 v2 — 메인(랜딩) 페이지
> **산출물**: PC 정적 목업 ~4장 (비로그인 + 쇼케이스 + 로그인 + 컬러 팔레트)
> **관련 문서**: GNB 하위 메뉴 구조 요청서 (`design-request-gnb-submenu.md`) — 전체 메뉴구성도·화면 구조도 포함

---

## 1. 페이지 목적

지자체 CCTV 재난·사고 영상으로 구축한 AI 학습데이터를 기업·연구자에게 제공하는 포털의 **첫 진입점**.
데이터 검색으로의 빠른 진입, 서비스 규모감 전달, 최신 소식 노출이 핵심 역할.

---

[로그인]   로고(AI 영상학습 사용자 포털) | ──GNB── | 사이트맵(≡)  로그아웃  마이페이지

## 4. 섹션 구성 + 샘플 데이터

> 기본 순서: 빠른 검색(목업 캐릭터 활용) → 보유 현황 → 이벤트 유형별 쇼케이스 → 인기 데이터셋 → 최근 공지사항 → 서비스 소개 배너
> 디자이너 판단으로 순서 변경·통합 제안 가능
> **※ 아래 모든 수치는 목업용 샘플 데이터이며 실제 운영 데이터와 무관**

### 섹션 1: 빠른 검색 (히어로 영역)

**역할**: 메인 CTA. 학습데이터 검색으로의 가장 빠른 진입점.

| 요소                    | 샘플                                                                                            |
| ----------------------- | ----------------------------------------------------------------------------------------------- |
| 타이틀                  | AI 영상학습 데이터, 검색부터 시작하세요                                                         |
| 서브타이틀              | 재난·안전 CCTV 영상 기반 학습데이터를 검색하고 활용하세요                                      |
| 검색바 placeholder      | 데이터셋명, 키워드로 검색                                                                       |
| 카테고리 칩 (빠른 필터) | `쓰러짐` `폭력` `화재` `교통사고` `유괴` `침수` `산불` `산사태` `전체보기 →` |

**디자인 노트**:

- 페이지에서 가장 눈에 띄는 영역. 배경 이미지/그라데이션 활용 가능
- 검색바는 충분히 크고 명확하게
- 카테고리 칩은 클릭 시 해당 카테고리로 검색 결과 페이지 이동

---

### 섹션 2: 보유 현황 (수치 카드)

**역할**: 서비스 규모감을 수치로 전달. 클릭 시 소식&참여 > 데이터 현황으로 이동.

| 카드   | 라벨          | 샘플 수치 | 단위 |
| ------ | ------------- | --------- | ---- |
| 카드 1 | 학습 데이터셋 | 1,247     | 건   |
| 카드 2 | 영상 클립     | 38,520    | 건   |
| 카드 3 | 라벨링 프레임 | 524,300   | 장   |
| 카드 4 | 누적 다운로드 | 12,840    | 회   |

**디자인 노트**:

- 4개 카드 가로 배치 (아이콘 + 수치 + 라벨)
- 수치는 크게, 단위는 작게
- 운영 시 카운트업 애니메이션 적용 예정이나 목업에서는 정적
- "자세히 보기 →" 링크 → 소식&참여 > 데이터 현황

---

### 섹션 3: 이벤트 유형별 데이터셋 쇼케이스

**역할**: 8종 재난·안전 이벤트 유형별 학습데이터 현황을 아이콘/카드 그리드로 시각화. 각 카드 클릭 시 해당 유형 필터가 적용된 검색 페이지로 이동.

| 유형     | 데이터셋 | 영상 클립 |
| -------- | -------- | --------- |
| 쓰러짐   | 124건    | 3,720건   |
| 폭력     | 98건     | 2,940건   |
| 화재     | 187건    | 5,610건   |
| 교통사고 | 203건    | 6,090건   |
| 유괴     | 45건     | 1,350건   |
| 침수     | 156건    | 4,680건   |
| 산불     | 112건    | 3,360건   |
| 산사태   | 122건    | 3,660건   |

**카드에 표시할 정보**:

- 유형 아이콘 (일러스트 또는 라인 아이콘)
- 유형명
- 데이터셋 N건 / 영상 클립 N건

**디자인 노트**:

- 4x2 그리드 또는 가로 스크롤 등 자유 배치
- 데이터가 0건인 유형도 카드 유지 (빈 유형도 노출)
- 각 카드 전체 클릭 가능 (해당 유형 필터 적용 검색 페이지 이동)
- 히어로 영역의 카테고리 칩과 동일한 이벤트 유형이지만, 여기서는 **통계 수치를 시각적으로 표현**하는 것이 핵심

### 섹션 4: 인기 데이터셋

**역할**: 다운로드 상위 데이터셋을 노출하여 탐색 유도. 클릭 시 학습데이터 상세로 이동.

| # | 데이터셋명                     | 카테고리 | 데이터 유형   | 건수    | 다운로드 |
| - | ------------------------------ | -------- | ------------- | ------- | -------- |
| 1 | 도심 화재 감지 영상 데이터셋   | 화재     | 영상 + 라벨   | 3,200건 | 1,842회  |
| 2 | 도로 교통사고 CCTV 학습데이터  | 교통사고 | 영상 + 라벨   | 5,100건 | 1,523회  |
| 3 | 공공시설 쓰러짐 감지 데이터셋  | 쓰러짐   | 이미지 + 라벨 | 2,800건 | 1,105회  |
| 4 | 하천 침수 모니터링 영상 데이터 | 침수     | 영상          | 1,950건 | 987회    |
| 5 | 산사태 위험구간 감시 데이터셋  | 산사태   | 영상 + 라벨   | 1,420건 | 756회    |
| 6 | 폭력 행위 탐지 학습 데이터셋   | 폭력     | 영상 + 라벨   | 4,300건 | 632회    |

**카드에 표시할 정보**:

- 썸네일 (영상 캡처 이미지 placeholder)
- 데이터셋명
- 카테고리 뱃지 (쓰러짐/폭력/화재/교통사고/유괴/침수/산불/산사태)
- 데이터 유형 태그 (영상/이미지/라벨)
- 다운로드 수
- 즐겨찾기 아이콘 (♡)

**디자인 노트**:

- 카드 4~6개. 한 줄 4개 또는 3+3 두 줄 등 자유
- "전체 데이터셋 보기 →" 링크
- 썸네일은 목업에서 placeholder 이미지 (회색 박스 + 카테고리 아이콘)

### 섹션 6: 서비스 소개 배너

**역할**: 포털 핵심 기능 3가지를 시각적으로 어필.

| # | 제목            | 설명                                                                              | CTA                  |
| - | --------------- | --------------------------------------------------------------------------------- | -------------------- |
| 1 | 학습데이터 검색 | 재난·안전 CCTV 영상 기반 AI 학습데이터를 다양한 조건으로 검색하고 다운로드하세요 | 데이터 검색하기 →   |
| 2 | 저작도구        | 업로드한 영상에 직접 마킹·라벨링하여 나만의 학습데이터를 만들어 보세요           | 저작도구 시작하기 → |
| 3 | AI 데이터 증강  | AI 기술로 학습데이터를 증강하고 생성형 AI로 새로운 영상을 만들어 보세요           | AI 증강 알아보기 →  |

**디자인 노트**:

- 3개 카드 또는 가로 배너 형태
- 각 카드에 아이콘/일러스트 + 제목 + 설명 + CTA 버튼
- 캐러셀이지만 목업에서는 3개 동시 노출 (정적)
