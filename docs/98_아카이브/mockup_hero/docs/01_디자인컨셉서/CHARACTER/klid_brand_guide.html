Klid(가칭)
데이터의 가능성을 연결하고,
더 큰 가치를 만드는 클리드의 데이터 파트너

Klid는 작은 데이터 조각에서 시작해 연결하고, 확장하고,
새롭게 만들어가며 AI 학습데이터의 무한한 가능성을 성장시킵니다.
움직이는 작은 다리와 팔로 세상을 탐색하고,
주변에 데이터를 연결해 가치를 만들어갑니다.

연결다양한 데이터를 안전하게 연결합니다.
확장AI 기술로 데이터를 증강하고 확장합니다.
생성생성형 AI로 새로운 데이터를 만듭니다.
활용더 큰 가치를 위해 데이터를 활용합니다.
