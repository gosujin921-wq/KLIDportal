Design Concept

AI 영상학습
사용자 포털

포털의 핵심 가치를 두 가지 비주얼 방향으로 제안합니다. 데이터 자체를 주인공으로 삼은 A안, 데이터 경험을 친근하게 안내하는 B안.

Business Value

이 포털은 데이터를 제공하는 곳을 넘어,
데이터를 확장하고 활용하는 플랫폼입니다

AI 영상학습 사용자 포털은 원본 영상·이미지 데이터를 단순 조회하거나 다운로드하는 서비스가 아닙니다. 사용자는 필요한 데이터를 탐색하고, AI 증강과 생성 기술을 통해 더 많은 학습데이터로 확장하며, 프로젝트 목적에 맞게 활용할 수 있어야 합니다.

From

데이터 제공

정해진 데이터를 검색하고 내려받는 기존 포털형 경험.

To

데이터 확장

원본 데이터가 증강·생성을 거쳐 더 큰 학습 자산으로 성장하는 경험.

Value

데이터 활용

확장된 데이터를 연구·서비스·모델 학습에 연결하는 실질적 활용 경험.

Brand Message

DATA TO VALUE
데이터를 가치로

흩어진 원천 데이터를 연결하고, AI로 확장하며, 새로운 학습데이터로 성장시키는 연결형 데이터 생태계.

CONNECT
연결

영상·이미지·라벨·메타데이터를 하나의 활용 흐름으로 연결합니다.

EXPAND
확장

AI 증강을 통해 데이터의 수량과 조건, 활용 범위를 넓힙니다.

CREATE
생성

생성형 AI를 통해 부족한 학습데이터를 새롭게 만들어냅니다.

Design Direction

데이터의 가치를 전달하는
두 가지 브랜드 경험 방향

Concept A
Value in Structure
데이터가 가치로 성장하는 과정을 시각화

포털의 핵심 자산인 데이터를 직접 주인공으로 삼아, 하나의 원본 데이터가 연결·증강·생성을 거치며 더 큰 데이터 구조로 성장하는 과정을 시각화합니다.

데이터 중심확장성신뢰감전문성
Concept B
Value in Experience
데이터 활용 경험을 캐릭터로 전달

데이터 셀(Data Cell)을 의인화한 캐릭터로, 데이터의 연결·확장·생성 과정을 친근하게 전달하는 브랜드 파트너.

캐릭터 중심접근성친근함서비스성
Concept A

EXPAND
데이터가 확장되는 플랫폼

하나의 큐브

원본 데이터

지자체 CCTV 클립영상 한 건. 영상과 이미지가 묶인 한 세트.

하나에서 여럿으로

증강 · 생성

생성형 AI와 증강으로 학습데이터가 늘어남. 포털의 핵심 가치.

색·크기의 다양성

이벤트 · 조건

산불·침수·유괴 등 이벤트와 날씨·시간대 조건별로 갈라지는 데이터셋.

메탈릭·하나로 회귀

통합 데이터 자산

증강·생성·가공된 데이터가 하나의 공공 데이터 자산으로 축적됨을 의미.

Concept A · Message

데이터를 보여주는 것이 아니라,
데이터가 가치로 성장하는 과정을 보여줍니다

Connect

연결

흩어진 데이터 조각을 하나의 구조로 엮습니다.

Expand

확장

데이터의 조건과 활용 범위를 넓힙니다.

Generate

생성

부족한 데이터를 AI로 새롭게 구축합니다.

Activate

활용

확장된 데이터를 실제 학습과 연구에 연결합니다.

Concept B

GUIDE
데이터 경험을 안내하는 파트너

B안은 데이터의 최소 단위가 살아난 존재로서, 포털의 기능과 데이터 성장 과정을 친근하게 전달합니다.

  • 1
    데이터 셀

    작은 데이터 조각에서 시작하는 학습데이터의 기본 단위를 캐릭터화합니다.

  • 2
    서비스 안내자

    탐색·증강·생성·분석 흐름을 딱딱한 기능 설명이 아니라 행동과 표정으로 전달합니다.

  • 3
    브랜드 자산

    히어로, 빈 상태, 로딩, 완료 안내 등 포털 전반의 사용자 접점을 부드럽게 연결합니다.

Concept B · Message

기술을 귀엽게 만드는 것이 아니라,
복잡한 데이터 과정을 쉽게 이해시키는 것입니다

Explore

탐색

필요한 데이터를 발견하고 선택하는 경험을 안내합니다.

Enhance

증강

원본 데이터를 더 다양한 조건으로 확장하는 과정을 보여줍니다.

Create

생성

새로운 학습데이터가 만들어지는 장면을 직관적으로 전달합니다.

Trust

신뢰

공공 데이터 기반의 안정성과 지원 이미지를 부드럽게 보완합니다.
